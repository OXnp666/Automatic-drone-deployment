本项目为无人机自动投放，技术要点如下:1.机身采用F450机型，飞控采用Pixahwk2.4.8飞控（即stm32f427搭配5611气压计），无人机飞控软件为Ardupilot,配套地面站为MissionPlanner2.机载电脑为树莓派4b，内置安装了ubuntu20.04系统3.配备LQ图数传2.4G，可实现3000米的稳定图传4.搭载光流模块和激光定高模块可以实现室内室外无GPS信号环境下的定位导航5.云台相机采用思翼A8mini搭配图数传模块机械臂技术：1.机身采用阳极氧化铝铝合金坚固可靠耐腐蚀2.采用JETSON NANO为主控，Opencv为图像处理库，使用Python3.内置ROS操作系统，采用逆运动学算法分解机械臂舵机运动控制4.可进行机械臂Movelt拟真控制
无人机技术，1.机身采用F450机型，飞控采用Pixahwk2.4.8飞控（即stm32f427搭配5611气压计），无人机飞控软件为Ardupilot,配套地面站为MissionPlanner2.机载电脑为树莓派4b，内置安装了ubuntu20.04系统，使用4b的wifi开机自建热点，PC端可直连4b网络组建局域网，通过ssh命令远程连接树莓派3.配备LQ图数传2.4G，可实现3000米的稳定图传4.搭载光流模块和激光定高模块可以实现室内室外无GPS信号环境下的定位导航5.云台相机采用思翼A8mini搭配图数传模块可实现稳定的4K画面录制，配备星光级传感器搭配超强感光性能CMOS，低照度环境下也可获得清晰画面
机械臂技术：1.机身采用阳极氧化铝铝合金坚固可靠耐腐蚀2.采用JETSON NANO为主控，Opencv为图像处理库，使用Python为主要编程语言3.内置ROS操作系统，采用逆运动学算法分解机械臂6个自由度的舵机运动控制，通过输入目标坐标计算各舵机的理论运动角度，结合舵机控制协议同时控制各个舵机运动4.可进行机械臂Movelt拟真控制，可自定义学习动作组5.机械臂搭载USB摄像头，拥有30万像素，110度视场角，480p分辨率为opencv的视觉分析提供硬件支撑
整体机技术：1.遥感测绘功能，通过相机定点抓拍照片，可通过大疆智图实现地形图绘制2.热成像搜寻：搭载640×512/13 mm 热成像相机，可实现全局测温，搭载思翼AI跟踪模块可实现对目标实时跟随与捕捉，具有高对比度可实时搜寻3.避障雷达双模式，顶部搭载思蓝A1雷达可实现360度无障碍避障，底部搭载激光定高雷达为保证飞机安全执行提供强有力的保证4.搭载的机载电脑编写的Dronekit-python控制可实现无人机自主飞行，自主降落，自主返航5.机械臂可通过拟真手臂的转动，舵机按照实际需求进行夹持抓取操作，更好的进行救援任务
衍生机型：搭载360°激光雷达和抛投器进行快速投放
