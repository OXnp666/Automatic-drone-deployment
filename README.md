本项目为无人机自动投放，技术要点如下:1.机身采用F450机型，飞控采用Pixahwk2.4.8飞控（即stm32f427搭配5611气压计），无人机飞控软件为Ardupilot,配套地面站为MissionPlanner2.机载电脑为树莓派4b，内置安装了ubuntu20.04系统3.配备LQ图数传2.4G，可实现3000米的稳定图传4.搭载光流模块和激光定高模块可以实现室内室外无GPS信号环境下的定位导航5.云台相机采用思翼A8mini搭配图数传模块机械臂技术：1.机身采用阳极氧化铝铝合金坚固可靠耐腐蚀2.采用JETSON NANO为主控，Opencv为图像处理库，使用Python3.内置ROS操作系统，采用逆运动学算法分解机械臂舵机运动控制4.可进行机械臂Movelt拟真控制
